reglas y metodo de importación

preparando el archcivo a importar:
  • Todas las tablas de la carpeta se exportan a "Delimited Text" en paradox, las opciones de exportacion son:
    1. Character Set: ANSI
    2. Field Names: seleccionado
  • En la carpeta de trabajo de paradox, comprimir todos los .txt exportados en un zip, el zip debe tener el nombre de la carpeta de trabajo, este zip es el archivo a importar
importando:
  • El archivo a importar es el ultimo campo a seleccionar, cuando se halla seleccionado el formulario se envia automaticamente.
  • Si hay errores importando, estos se reflejan en la barra de progreso.
despues de importar:
  • Los veneficiarios al importarse se asignan por defecto a la primera cuenta bancaria existente(en la version anterior de vinci esta interface nunca se completó), asi q luego de importar deben reasignarce a sus verdaderas cuentas
ademas:
  • El nombre del zip es el nombre de la contabilidad importada, se usa ademas para determinar las dependencias en los casos de contabilidades consolidadas, una vez importadas todas las carpetas de trabajo se podran cambiar los nombres de las contabilidades, pero si se hace antes, entonces hay q reasociar los origenes para las contabilidades consolidadas
  • Cuando se importa una carpeta el sistema hace un respaldo de datos de forma automatica, tengalo en cuenta en caso de q hallan errores de importación
